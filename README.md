# SimpleCloud
Fast and simple file manager which support all operations in Dropbox and GoogleDrive

The following features are supported:
- Breadcrumb navigation
- Efficient multi-threading -> speed
- Basic file operations (copy/cut/paste/rename/create/zip/properties/share)
- Basic Dropbox and Google Drive operations (view/copy/cut/paste/rename/create/download/upload/properties/share)
- Bookmark
